# 量子理論の基礎

量子コンピュータを理解するためには、その動作原理である量子力学の基礎的な性質を把握しておく必要があります。
ここでは、超伝導量子回路を扱う上で最低限必要となる概念と数学的記法について概説します。

## 量子ビット (Qubit) とは

従来のコンピュータ（古典コンピュータ）では、情報は「0」か「1」のいずれかの状態を取る**ビット (bit)** で表現されます。
一方、量子コンピュータでは、**量子ビット (qubit)** と呼ばれる情報の最小単位を用います。

量子ビットは、「0」の状態 ($|0\rangle$) と「1」の状態 ($|1\rangle$) の**重ね合わせ (superposition)** 状態を取ることができます。
これは、ある確率で0が観測され、ある確率で1が観測されるような状態です。

$$
|\psi\rangle = \alpha |0\rangle + \beta |1\rangle
$$

ここで、$\alpha$ と $\beta$ は複素数であり、確率振幅と呼ばれます。観測確率は $|\alpha|^2$ と $|\beta|^2$ で与えられ、$|\alpha|^2 + |\beta|^2 = 1$ を満たします。

## 重要な量子現象

### 1. 重ね合わせ (Superposition)
複数の状態が同時に存在しているような状態です。波の重ね合わせと同様に、量子状態も線形結合で表されます。これにより、並列計算のような効果が期待できます。

### 2. 量子もつれ (Entanglement)
2つ以上の量子ビット間に存在する特殊な相関関係です。一方の量子ビットの状態を測定すると、他方の状態も瞬時に確定します。この性質は、量子テレポーテーションや量子誤り訂正において重要な役割を果たします。

### 3. 干渉 (Interference)
確率振幅は複素数であるため、正の振幅と負の振幅が打ち消し合う（破壊的干渉）ことや、強め合う（建設的干渉）ことが起きます。量子アルゴリズム（グローバーの探索アルゴリズムなど）では、この干渉を巧みに利用して、正解の状態の確率振幅だけを増幅させます。

## 数学的枠組み：ブラ・ケット記法 (Bra-Ket Notation)

量子力学では、ポール・ディラックによって導入された**ブラ・ケット記法**が広く用いられます。

- **ケットベクトル ($| \psi \rangle$)**: 列ベクトルを表します。量子状態記述します。
  $$
  |0\rangle = \begin{bmatrix} 1 \\ 0 \end{bmatrix}, \quad |1\rangle = \begin{bmatrix} 0 \\ 1 \end{bmatrix}
  $$

- **ブラベクトル ($\langle \psi |$)**: ケットベクトルの随伴（転置複素共役）を表します。行ベクトルになります。
  $$
  \langle \psi | = |\psi\rangle^\dagger = (\alpha |0\rangle + \beta |1\rangle)^\dagger = \alpha^* \langle 0 | + \beta^* \langle 1 |
  $$

- **内積 ($\langle \phi | \psi \rangle$)**: 2つの状態の重なり具合（スカラー値）を表します。正規直交基底であれば、$\langle 0 | 0 \rangle = 1, \langle 0 | 1 \rangle = 0$ となります。
