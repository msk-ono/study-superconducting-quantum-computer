# 超伝導量子コンピュータの概要

超伝導量子コンピュータは、超伝導回路を用いて量子ビット (qubit) を実現する方式です。
巨視的な量子状態を利用するため、デザインの自由度が高く、既存の半導体プロセス技術を応用できる可能性があるため、現在最も開発が進んでいる方式の一つです。

## 特徴

- **デザインの自由度が高い**: 回路パラメータ (インダクタンス、キャパシタンス) を設計することで、量子ビットの周波数や結合の強さを調整できます。
- **ゲート操作が高速**: マイクロ波パルスを用いて、ナノ秒オーダーでの高速なゲート操作が可能です。
- **コヒーレンス時間**: 外部環境との結合によりデコヒーレンスが起きやすいですが、近年は材料科学や回路設計の進歩により大幅に改善されています。

## Transmon Qubit

現在、多くの超伝導量子コンピュータ (IBM, Google, Rigetti等) で採用されているのが **Transmon (Transmission-line Shunted Plasma Oscillation Qubit)** 型の量子ビットです。
Transmon は、ジョセフソン接合に大きなシャウント容量を並列に接続することで、電荷ノイズに対する感度を劇的に低下させたものです。

本ドキュメントでは、主にこの Transmon 型量子ビットを対象として解説します。
